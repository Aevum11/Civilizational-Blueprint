## Part XV: Troubleshooting Guide

### If the device doesn't work:

**Problem:** No discharge at all

```
Check:
1. Battery voltage (should be 3.7V nominal)
2. Oscillator running (measure with scope/meter)
3. Connections to coil (continuity test)
4. Tip electrode not shorted to cavity

Fix:
- Replace battery
- Verify oscillator circuit
- Resolder connections
- Check for internal shorts
```

**Problem:** Very weak discharge (< 1mm arc)

```
Possible causes:
1. Cavity not resonant at 500 MHz
2. Coil material poor conductor
3. Diodes wrong polarity
4. Tip not sharp enough

Fix:
- Measure cavity with antenna/VNA
- Verify coil DC resistance < 10 Ω
- Check diode orientation
- Resharpen tip, aim for <100 μm
```

**Problem:** Discharge happens but immediately stops

```
Causes:
1. Insufficient charge storage (need capacitor)
2. Battery can't supply current fast enough
3. Tip oxidizing (resistance increasing)

Fix:
- Add small capacitor (100pF-1nF) at tip
- Use higher current battery
- Clean tip with fine sandpaper
```

**Problem:** Erratic behavior, random discharges

```
Causes:
1. Oscillator instability
2. Electromagnetic interference
3. Moisture/contamination

Fix:
- Add ferrite bead to power lines
- Shield cavity with aluminum
- Dry out device, seal openings
```
